# Baigiamasis_darbas
Šis projektas yra reverse image search. Jame naudojamas apmokintas Resnet-50 modelis kuris yra apmokintas ant imageNet duomenų. Surasti artimiausius kaimynus naudojamas NearestNeighbours metodas.
